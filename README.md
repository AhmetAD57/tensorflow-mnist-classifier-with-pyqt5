# Tensorflow MNIST Classification with PyQt5
<img src="./images/banner1.png"/>

## Used Tools
- TensorFlow
- PyQt5
- NumPy
- OpenCV
- Matplotlib
- seaborn

## Quick Installation
```
pip install -r requirements.txt
```
## Run Project
```
python mnist_classifier_ui.py
```

## Features
- Testing from model test set or external image.
- User friendly GUI.
- Digit finding and cuting.
